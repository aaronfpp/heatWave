# heatWave.exe Quick Start Guide for Coaches

## ✅ Installation (5 seconds)

1. **Download** `heatWave.exe` (or `heatWave.zip` and extract it)
2. **Move to Desktop** (or any location you prefer) - optional
3. **Done!** No installation needed

## ▶️ Running heatWave

### First Time:
1. **Double-click** `heatWave.exe`
2. **Wait 5-10 seconds** - the app is starting
3. **Your browser opens** automatically to the heat sheet generator
4. **Start using it!** Upload your psych sheet PDF

### Subsequent Times:
- Just double-click `heatWave.exe` again
- It opens in your browser automatically

## 🏊 Using heatWave

### Step 1: Upload Psych Sheet
- Click "Upload" tab
- Drag & drop your USA Swimming psych sheet PDF
- Click "Parse PDF" button
- Wait for processing (usually <5 seconds)

### Step 2: Check Preview
- Click "Preview" tab
- Verify your events and entries look correct
- Use filters to search/navigate if needed

### Step 3: Customize Settings
- Click "Settings" tab
- Edit meet name (e.g., "State Championship")
- Set date (format: MM/DD/YYYY)
- Confirm pool lanes (usually 8)

### Step 4: Generate Heat Sheets
- Click "Generate" tab
- Click "Generate Heat Sheets" button
- Wait for seeding to complete (~3-5 seconds)

### Step 5: Download PDFs
- **Full Meet PDF** - All events in one file (best for meet director)
- **Individual Event PDFs** - One file per event (for timers)
- Click the download button(s)

### Step 6: Print & Use
- Open the PDF files
- Print for meet day
- Distribute to officials, timers, and other coaches

## 🎯 Tips & Tricks

### Upload Different Meets
- Just upload a different psych sheet
- Each upload starts fresh (no data is saved)
- You can process as many meets as you want

### Multiple Lanes
- Adjust "Lanes" setting in Settings tab
- Default is 8 (standard 25-yard pool)
- Use 6 for smaller pools, 10 for Olympic pools

### Issue: Browser Doesn't Open
- Look for `http://localhost:8501` in your address bar
- Type it manually if browser didn't open

### Issue: Takes Too Long to Start
- First run takes ~10-15 seconds (normal)
- Subsequent runs are faster
- If stuck after 30 seconds, close and try again

## 🔒 Privacy & Security

✅ **Your data is safe:**
- All processing happens on your computer
- No internet connection required
- No PDFs or data saved to disk
- Nothing uploaded to servers
- Completely anonymous and private

## ⚙️ Troubleshooting

### "File Not Found" Error
- Make sure heatWave.exe is in the same folder as other files
- If you downloaded a ZIP, make sure to extract everything

### Browser Shows "Connection Refused"
- Close heatWave.exe completely
- Wait 5 seconds
- Double-click heatWave.exe again
- Wait for browser to open

### "Cannot read PDF" Error
- Verify you're uploading a USA Swimming psych sheet
- Make sure it's a standard text-based PDF (not scanned image)
- Try downloading the psych sheet again from your meet software

### Application Won't Start
- Restart your computer
- Download a fresh copy of heatWave.exe
- Ensure you have ~200 MB of free disk space
